# Image_To_Loading_GIF
Convert images to loading GIF with simple python script


## How to ?
how to change the final gif size? from here:

`canvas_size = max(1000, 1000)`

how to play with speed and smoothness? from here:

`frame_count = 48`

`duration = 30`
